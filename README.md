# Booking App

This a command line application that books a squash court.

## Usage

```bash
./booker -c 1 -d 20 -h 19 -m 50 -t 17
```
Flag | Description |
--- | ---
-c|  court to book  
-d|  days in the future to book it  
-h|  hour of court in 24hr format  
-m|  minute of court in 24hr format  
-t|  timeslot of court
